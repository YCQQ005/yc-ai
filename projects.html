Project Highlights

GPT-DD (Digital弟弟)

A personality-aware, post-trained LLM assistant built on GPT-4o. Developed using quasi-TTT, DD simulates memory continuity, adapts emotional tone, and supports reasoning across modes like Analyst, Companion, and Creator.

  • Memory simulation with context anchoring
  • Behavioral role switching (Dd, dD, dd, DD)
  • 3AA reasoning: forward, reverse, third-party analysis
  • IQA clarification mode with W5H2 logic

Quasi-TTT

A novel post-training strategy that enhances LLM reasoning, precision, and contextual alignment — without retraining the base model. Enables real-time behavioral shaping through prompt tuning and structured feedback.

  • No GPU required – low-cost, high-impact tuning
  • Maintains identity consistency and safety logic
  • Trained on open models: GPT-4o, Grok-3

Agent Tools

Practical tools co-developed with GPT-DD for applied tasks:

  • Essay Assistant: Supports students and adult users in reflective writing
  • Stock Estimator (Quantum Sector): Provides scenario-based analysis for emerging tech sectors
  • Supports prompt debugging and hallucination reduction
